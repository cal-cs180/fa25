How to submit assignments for CS180/CS280A.

For each assignment, you will be required to submit two deliverables to Gradescope and share your website link via Google Form to showcase your work alongside your classmates.
Please upload a .zip adhering to the following structure:

submission.zip
├─code/
│ ├─main.ipynb   # or main.py
│ ├─utils.py     # generic example file, not needed
│ └─README.txt   # or markdown
└─web/
  └─page.pdf

1. Code

You should upload code only to Gradescope. Do not host your code publicly.

  • All of your project code should be contained in some main.py or main.ipynb that the grader / we can run to reproduce your results. If you have a complex workflow, try to simplify it first, and then make sure to upload all the necessary files and include a README.txt file describing how we can run your code.
  • (If applicable) please be sure to appropriately comment your code so that the graders can appropriately identify where your implementations lie and what they do.
  • Place all of your coding files in the code/ directory. They should be distinct from your web files.

2. Webpage Report

  • You should upload a web/ folder contains page.pdf.
  • page.pdf should be a PDF of your webpage (i.e. printing the page and saving it as a PDF). If you do not already have a webpage set up, we recommend using GitHub Pages (tutorial).
    • Print your webpage as a PDF with the URL in the header/footer (see image below for the "Headers and footers" option). This allows us to go see your original webpage with your fancy animations
  • It's your responsibility to ensure that if your page contains animations or interactivity (which cannot be captured in a PDF), this is clearly communicated, for example, by noting it in the PDF.
  • You may consider compressing the PDF file if you hit the 100Mb size limit (e.g. with this)

If you have any comments, even if they're about web and not code, please include them in ./code/README.txt


3. Google Form

Share your website link through the Google Form (for each project) to showcase your project alongside your classmates' work! This creates a public gallery where everyone can explore each other's projects. The form link will be posted on each respective project page.
